# 🚛 Logistics Data Pipeline

A Python-based data pipeline that cleans, processes, and analyzes
last-mile delivery route data to identify inefficiencies and
optimize distribution patterns.

---

## 🧠 Problem Statement

Rural municipalities in Alberta face significant last-mile delivery
gaps — sparse populations, long distances, and no centralized parcel
infrastructure. This pipeline processes raw delivery data to surface
route inefficiencies and support hub placement decisions.

---

## ⚙️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
